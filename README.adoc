= LLM Router Example

This is a Spring AI and Spring Integration application that routes customer questions to the correct agent.

== Requirements:

* Java 17 or Above

== Build:

[source,shell]
----
export OPENAI_API_KEY=<Your Open AI Key>
mvn clean package
----

== Run:

[source,shell]
----
export OPENAI_API_KEY=<your Open AI Key>
java -jar target/llmRouterFlow-0.0.1-SNAPSHOT.jar
----
